# قائمة قبول الـPilot بعد النشر

## البيئة والحسابات

- [ ] يعمل رابط Vercel عبر HTTPS.
- [ ] متغيرات البيئة الأربعة موجودة.
- [ ] يستطيع المدير تسجيل الدخول.
- [ ] لا يستطيع مستخدم غير مرتبط فتح لوحة الإدارة.
- [ ] يمكن إنشاء موظف استقبال وفني واختبار صلاحياتهما.

## العملاء والسيارات

- [ ] إضافة عميل وتعديله والبحث عنه تعمل.
- [ ] يمنع تكرار رقم الجوال داخل الورشة.
- [ ] أرشفة العميل لا تحذف سجلاته السابقة.
- [ ] إضافة سيارة وربطها بالعميل تعمل.
- [ ] تعديل السيارة والبحث باللوحة أو VIN يعمل.
- [ ] أرشفة السيارة لا تحذف تاريخها.

## الخدمات

- [ ] إضافة خدمة وتعديل السعر والمدة يعملان.
- [ ] إيقاف خدمة يخفيها من الموقع والحجز.
- [ ] تظهر القيم المحدثة في `/services` و`/book`.

## الحجوزات وأوامر العمل

- [ ] إنشاء حجز من الجوال يعمل.
- [ ] يتحول الجوال السعودي إلى `+9665xxxxxxxx`.
- [ ] يظهر الحجز في لوحة الإدارة.
- [ ] لا يتكرر العميل عند استخدام الجوال نفسه.
- [ ] لا يتكرر أمر العمل عند الضغط مرتين.
- [ ] الانتقالات المسموحة تعمل وغير المسموحة تُرفض.

## التشخيص

- [ ] إنشاء تقرير مرتبط بأمر العمل يعمل.
- [ ] حفظ أكواد DTC المتعددة يعمل.
- [ ] تُحفظ النتائج والتوصيات والخلاصة.
- [ ] لا يظهر التقرير للعميل افتراضيًا.
- [ ] يظهر بعد تفعيل `customer_visible`.

## الموافقات

- [ ] إنشاء طلب موافقة بالمبلغ وملخص الأعمال يعمل.
- [ ] ينتقل أمر العمل إلى `waiting_approval`.
- [ ] لا يمكن إنشاء طلبين معلقين لنفس الأمر.
- [ ] يستطيع الموظف تسجيل موافقة أو رفض مع القناة والملاحظة.
- [ ] يستطيع العميل رؤية الطلب المسموح به والرد من البوابة.
- [ ] تنقل الموافقة أمر العمل إلى `approved`.

## الجودة

- [ ] لا يمكن الانتقال مباشرة من `in_progress` إلى `ready`.
- [ ] يمكن الانتقال إلى `quality_check`.
- [ ] حفظ مسودة قائمة الجودة يعمل.
- [ ] لا يمكن اعتماد النجاح مع بند معلق أو فاشل.
- [ ] الفشل يعيد أمر العمل إلى `in_progress`.
- [ ] النجاح ينقل أمر العمل إلى `ready` ويسجل `completed_at`.

## المستندات

- [ ] رفع PDF أو صورة أو CSV ضمن 15 MB يعمل.
- [ ] النوع أو الحجم غير المسموح يُرفض.
- [ ] ربط سجل يخص عميلًا آخر يُرفض.
- [ ] المستند المخفي لا يظهر للعميل.
- [ ] المستند الظاهر يفتح برابط موقّت.
- [ ] حذف المستند متاح للمدير فقط.

## بوابة العميل

- [ ] يستطيع المدير تفعيل البوابة من صفحة العميل.
- [ ] تظهر كلمة المرور المؤقتة مرة واحدة.
- [ ] يستطيع العميل تسجيل الدخول.
- [ ] يستطيع تغيير كلمة المرور.
- [ ] يرى بياناته فقط.
- [ ] يرى السيارات والحجوزات والأوامر والموافقات والتقارير والمستندات المصرح بها.

## تجربة الجوال

- [ ] `/book` مناسب لـAndroid وiPhone.
- [ ] شريط التنقل لا يغطي الأزرار.
- [ ] صفحات الإدارة قابلة للاستخدام من الجوال.
- [ ] النوافذ والنماذج لا تتجاوز الشاشة.
